Fonctions

u32 PA_InitGBFS (void)
 Initialise GBFS et récupère des infos sur les fichiers. Renvoie le nombre de fichiers trouvés... Stock e les infos sous la forme suivante : PA_GBFSfile[numéro du fichier].File est un pointeur sur le fichier en question, .Name est son nom sans extension, .Ext est l'extension (txt, html, etc...), et .Length la taille du fichier.
s32 PA_GetGBFSFile (s16 start, char *name, char *extension)
 Chercher un fichier donné dans GBFS. Renvoie le numéro du fichier dans le systeme PA GBFS, ou -1 si le fichier n'a pas été trouvé.
s16 PA_GBFSSearchExt (s16 *array, char *extension)
 Cherche tous les fichiers de GBFS avec l'extension trouvée, place le numéro dans le tableau, et renvoie le nombre de fichiers trouvés...
